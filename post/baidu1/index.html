百度网盘满速技巧手机篇（1）：Termux+BaiduPCS

2019-07-20

此方法是使用Termux+BaiduPCS的Web进行下载百度网盘文件，更多使用方法请参考原项目：BaiduPCS。

下面是正式教程：

  1. 下载高级终端Termux
  2. 打开Termux，等待install（尽量在网络良好的条件，可能需要“梯子”），完成了界面会出现英文
  3. 切换为清华源（此处可选，主要是安装wget时快一点），过程要输入Y，完成这步要结束Termux后台重新进入软件开始第4步
sed -i 's@^\(deb.*stable main\)$@#\1\ndeb
https://mirrors.tuna.tsinghua.edu.cn/termux stable main@' $PREFIX/etc/apt/sources.list
apt update && apt upgrade
  1. 安装wget
    pkg install wget
  2. 下载压缩包（下载有点慢，请耐心等待，保持Termux运行，如果中断要清除Termux数据重来）
    wget https://github.com/liuzhuoling2011/baidupcs-web/releases/download/3.6.8/BaiduPCS-Go-3.6.8-android-arm64.zip
  3. 解压压缩包
    unzip BaiduPCS-Go-3.6.8-android-arm64.zip
  4. 给软件读取存储权限
    termux-setup-storage
  5. 运行
    ./BaiduPCS-Go-3.6.8-android-arm64/BaiduPCS-Go
  6. 打开浏览器（建议使用Chrome浏览器），地址栏输入localhost:5299，登录百度账号即可进行文件下载
  • 注意:下载过程中不能结束Termux的后台，下次使用只要没有清除掉Termux的数据就可以直接在Termux输入./BaiduPCS-Go-3.6.8-android-arm64/BaiduPCS-Go就可以接着第8步使用了

此方法要登录百度网盘账号，下载过多可能会被拉“小黑屋”（账号限速），此方法可能就会短暂失效。不过不用担心，被拉“小黑屋”后，等待几天或者直接开百度网盘会员，下载速度应该就会恢复。

此方法建议用于临时紧急下载百度网盘文件或者会员下载提速仍然很慢的时候，若在百度网盘有比较大的下载需求，建议支持正版百度网盘并开通百度网盘会员。
